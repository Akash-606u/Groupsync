## Features  
- User authentication (signup / login)  
- Create or join groups  
- Real‑time updates across group members  
- Simple and intuitive UI for managing groups  
- Server side API for group logic and collaboration  
- Modular frontend and backend folders for easier maintenance  

## Tech Stack  
- **Frontend**: React (or other JS framework)  
- **Backend**: Node.js + Express  
- **Database**: (e.g., MongoDB / PostgreSQL) – configure in server folder  
- **Communication**: HTTP API and/or WebSockets for real‑time events  
- **Other**: JavaScript, HTML, CSS
